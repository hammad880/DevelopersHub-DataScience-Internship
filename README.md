# DevelopersHub Data Science & Analytics Internship

This repository contains completed tasks for the Data Science & Analytics Internship at DevelopersHub Corporation.

## Completed Tasks

### Task 1: Exploring and Visualizing the Iris Dataset

Objective:
To understand dataset structure, perform data exploration, and create visualizations using pandas, matplotlib, and seaborn.

### Tools Used:
- Python
- Jupyter Notebook
- pandas
- matplotlib
- seaborn

### Key Insights:
- Petal length and width clearly separate iris species.
- Visualizations help identify relationships and data distribution.
